Cấu trúc Hệ thốngFrontend: HTML5, Tailwind CSS (Styling), Chart.js (Báo cáo), FullCalendar (Lịch).Backend: Supabase (Cơ sở dữ liệu & Xác thực).Giao diện: Quản lý tập trung qua file theme.css (Phong cách Apple Minimalism).PWA: Có khả năng cài đặt thành App trên iPad/Mobile qua manifest.json và sw.js.2. Nguyên tắc Logic Dữ liệu (Hard Validation)Hệ thống được thiết lập để đảm bảo dữ liệu nhập vào luôn logic:Success Calls $\le$ Total Calls.Booked Leads $\le$ Success Calls.Warm/Hot Leads không được vượt quá số lượng Success Calls.Dữ liệu được lưu trữ theo cấu trúc: Ngày -> Nhân viên -> Chi nhánh -> Số liệu.3. Hướng dẫn Cập nhật & Bảo trìThay đổi giao diện: Chỉnh sửa các biến màu tại :root trong file theme.css.Cập nhật App (PWA): Khi sửa code hoặc CSS, phải đổi tên CACHE_NAME trong sw.js (ví dụ từ v1 lên v2) để thiết bị người dùng cập nhật bản mới.Thêm nhân sự/chi nhánh: Thực hiện trực tiếp trên bảng staff hoặc branches trong Dashboard của Supabase.4. Quy trình cài đặt trên iPadTruy cập link bằng Safari.Nhấn nút Share (Chia sẻ).Chọn Add to Home Screen (Thêm vào màn hình chính).
